# 修改直播域名<a name="UpdateDomain"></a>

## 功能介绍

修改直播播放、RTMP推流加速域名相关信息

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Live&api=UpdateDomain)中调试该接口。

## URI

PUT /v1/\{project\_id\}/domain

## 请求示例

```
PUT https://{endpoint}/v1/{project_id}/domain

{
  "domain" : "play.example.huawei.com",
  "status" : "off"
}
```

## 响应示例

**状态码： 200**

直播域名修改成功

```
{
  "domain" : "play.example.huawei.com",
  "domain_type" : "pull",
  "domain_cname" : "play.example.huawei.com.c.cdnhwc3.com",
  "region" : "cn-north-1",
  "status" : "off",
  "create_time" : "2020-01-19T02:57:00Z",
  "domain_source" : {
    "protocol" : "rtmp",
    "source_type" : "huawei",
    "sources_priority" : "first",
    "sources" : [ "play.example.huawei.com.o.hwcloudlive.com" ]
  }
}
```

**状态码： 400**

参数错误

```
{
  "error_code" : "LIVE.100011001",
  "error_msg" : "Request Illegal"
}
```
