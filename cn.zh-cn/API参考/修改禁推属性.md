## 请求示例

```
PUT https://{endpoint}/v1/{project_id}/stream/blocks

{
  "domain" : "publish.example.huawei.com",
  "app_name" : "live",
  "stream_name" : "huawei",
  "resume_time" : "2018-05-19T21:10:15Z"
}
```
