## 请求示例<a name="section1636560210150255"></a>

```
POST /v1/{project_id}/record/config
 
{
  "domain": "push.hwcloud.cn",
  "app_name": "live",
  "record_duration": 900,
  "record_format": "flv",
  "record_location": "vod",
  "record_prefix": "{DomainName}/{AppName}/{StreamName}/{StartTime}-{EndTime}"
}

```
