# 提交录制控制命令<a name="RunRecord"></a>

## 功能介绍

对单条流的实时录制控制接口。

## 接口约束

在录制规则类型为COMMAND\_RECORD的时候有效。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Live&api=RunRecord)中调试该接口。

## URI

POST /v1/\{project\_id\}/record/control

## 请求示例

```
POST https://{endpoint}/v1/{project_id}/record/control?action=START

{
  "publish_domain" : "publish.xxx.com",
  "app" : "live",
  "stream" : "mystream"
}
```
