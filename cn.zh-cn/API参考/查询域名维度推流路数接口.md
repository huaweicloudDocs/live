# 查询域名维度推流路数接口<a name="ShowStreamCount"></a>

## 功能介绍

查询域名维度推流路数接口。

最大查询跨度31天，最大查询周期90天。

## 接口约束

该接口暂只支持查询“华北-北京四”区域，且部署在新版视频直播服务上的域名的相关数据。[如何查询域名是新版还是旧版？](https://support.huaweicloud.com/usermanual-live/live_01_0055.html)

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Live&api=ShowStreamCount)中调试该接口。

## URI

GET /v2/\{project\_id\}/stats/stream-count

## 请求示例

```
GET /v2/{project_id}/stats/stream-count?publish_domains=play.example.com&start_time=2020-06-04T07:00:00Z&end_time=2020-06-05T07:00:00Z

```

## 响应示例

**状态码： 200**

处理成功返回。

```
{
  "data_list" : [ {
    "time" : "2020-06-04T07:00:00Z",
    "value" : 0
  }, {
    "time" : "2020-06-04T07:05:00Z",
    "value" : 325
  } ]
}
```

**状态码： 400**

参数错误。

```
{
  "error_code" : "LIVE.100011001",
  "error_msg" : "Invalid request parameter: domain"
}
```
