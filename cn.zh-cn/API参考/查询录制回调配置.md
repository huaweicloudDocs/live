# 查询录制回调配置<a name="ShowRecordCallbackConfig"></a>

## 功能介绍

查询录制回调配置接口

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Live&api=ShowRecordCallbackConfig)中调试该接口。

## URI
