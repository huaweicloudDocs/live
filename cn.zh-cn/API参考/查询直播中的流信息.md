# 查询直播中的流信息<a name="ListLiveStreamsOnline"></a>

## 功能介绍

查询直播中的流信息

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Live&api=ListLiveStreamsOnline)中调试该接口。

## URI

GET /v1/\{project\_id\}/realtime/streams

## 请求示例

```
GET https://{endpoint}/v1/{project_id}/realtime/streams?publish_domain=push.example.huawei.com&app=live&offset=0&limit=10
```
