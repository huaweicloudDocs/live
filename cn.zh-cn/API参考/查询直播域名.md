# 查询直播域名<a name="ShowDomain"></a>

## 功能介绍

查询直播域名

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Live&api=ShowDomain)中调试该接口。

## URI
