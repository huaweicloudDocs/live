# 查询直播转码模板<a name="topic_retrieveTranscodingsTemplate"></a>

## 功能介绍

查询直播转码模板

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Live&api=ShowTranscodingsTemplate)中调试该接口，并获取该接口关联的CLI示例：  [hcloud Live ShowTranscodingsTemplate](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=Live&api=ShowTranscodingsTemplate)。

## 请求示例

```
GET https://{endpoint}/v1/{project_id}/template/transcodings?domain=play.example.huawei.com&app_name=live&page=0&size=10
```

## 响应示例

**状态码为 200 时: **

直播转码模板查询成功

```
{
  "domain" : "play.example.huawei.com",
  "total" : 1,
  "templates" : [ {
    "app_name" : "live",
    "quality_info" : [ {
      "PVC" : "off",
      "bitrate" : 4500,
      "codec" : "H264",
      "height" : 1080,
      "protocol" : "HLS",
      "quality" : "SD",
      "video_frame_rate" : 20,
      "width" : 1920
    }, {
      "PVC" : "off",
      "bitrate" : 4500,
      "codec" : "H264",
      "height" : 1080,
      "protocol" : "HLS",
      "quality" : "HD",
      "video_frame_rate" : 20,
      "width" : 1920
    } ]
  } ]
}
```

**状态码为 400 时: **

直播转码模板查询失败

```
{
  "error_code" : "LIVE.100011001",
  "error_msg" : "参数校验失败"
}
```
