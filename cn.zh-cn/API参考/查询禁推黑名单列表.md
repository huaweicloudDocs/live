    ```
    {
      "total": 2,
      "blocks": [
        {
          "app_name": "live",
          "stream_name": "test_001",
          "resume_time": "2018-05-19T21:10:15Z"
        },
        {
          "app_name": "live",
          "stream_name": "test_002",
          "resume_time": "2018-06-12T20:10:10Z"
        }
      ]
    }
    
    ```

-   处理失败返回（其他响应码，如返回400）

    ```
    {
      "error_code": "LIVE.100011001",
      "error_msg": "Request Illegal"
    }
    
    ```
