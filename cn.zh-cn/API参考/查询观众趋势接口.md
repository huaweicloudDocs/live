# 查询观众趋势接口<a name="ListUsersOfStreamV2"></a>

## 功能介绍

查询观众趋势。

最大查询跨度7天，最大查询周期90天。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Live&api=ListUsersOfStreamV2)中调试该接口，并获取该接口关联的CLI示例：  [hcloud Live ListUsersOfStreamV2](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=Live&api=ListUsersOfStreamV2)。

## URI

GET /v2/\{project\_id\}/stats/user

## 请求示例

```
GET /v2/{project_id}/stats/user?app=live&stream=pull_stream&play_domain=example.huawei.com

```

## 响应示例

**状态码为 200 时: **

处理成功返回。

```
{
  "data_list" : [ {
    "value" : 50,
    "time" : "2020-06-04T07:00:00Z"
  } ]
}
```

**状态码为 400 时: **

处理失败返回。

```
{
  "error_code" : "LIVE.100011001",
  "error_msg" : "请求参数非法：play_domain。"
}
```
