## 请求样例<a name="section2082716556150255"></a>

```
POST /v1/{project_id}/stream/blocks

{
  "domain": "play.example.com",
  "app_name": "live",
  "stream_name": "huawei",
  "resume_time": "2018-05-19T21:10:15Z"
}

```

    ```
      "error_msg": "Request Illegal"
    }
    
    ```
