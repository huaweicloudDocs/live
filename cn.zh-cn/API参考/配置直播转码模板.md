## 请求样例<a name="section156281393150255"></a>

```
PUT /v1/{project_id}/template/transcodings
{
  "domain": "play.example.com",
  "app_name": "live",
  "quality_info":
  [
    {
      "templateName": "",
      "quality": "lhd",
      "hdlb": "off",
      "PVC": "off",
      "codec": "H264",
      "width": 1920,
      "height": 1080,
      "bitrate": 2000,
      "video_frame_rate": 0,
      "protocol": "HLS",
      "iFrameInterval": 0
    }
  ]
}

```
